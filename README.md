# 拓饗國際網站

## 專案定位

拓饗國際網站是拓饗公司母體形象網站，負責說明集團定位、旗下事業與共享後勤架構。

目前網站包含：

- `/`：拓饗國際首頁
- `/buyba`：買吧直銷商城介紹與既有網站導流
- `/chiba`：吃吧餐飲連鎖介紹

## 目前範圍

- 已使用拓饗 LOGO。
- 已完成拓饗首頁。
- 已完成買吧分頁，並導向既有買吧本機頁面 `http://127.0.0.1:4173/?v=2`。
- 已完成吃吧分頁，定位為餐飲連鎖與標準化營運事業。
- 目前不使用 Supabase，因為尚無表單儲存、會員登入或後台資料需求。

## 本機啟動

```bash
npm install
npm run dev
```

本機網址：

```text
http://localhost:3000
```

## 建置檢查

```bash
npm run build
```

## 建議部署方式

第一階段建議使用：

- GitHub：管理網站原始碼與版本。
- Vercel：部署拓饗正式網站。
- Supabase：暫不需要。等需要合作表單、名單資料庫、會員登入、後台管理時再接。

## 後續可擴充

- 將買吧導流網址改為正式公開網址。
- 補上吃吧正式品牌素材、產品照片與合作洽詢表單。
- 新增合作表單，若要保存資料再接 Supabase。
- 接自訂網域，例如 `tuoxiang.com.tw` 或其他正式品牌網址。
